# Express Typescript Starter
https://github.com/anggardhanoano/express-ts-starter.git

Express Starter powered with Typescript and Sequelize

## Configure DB

- create .env file in project root directory
- add this into .env
  ```
  DATABASE_NAME=dbname
  USERNAME=username
  PASSWORD=password
  ```
